# Пример AI ответа для тестирования извлечения формул

## Входные данные (AI ответ):

```
1. Определяем коэффициенты

В квадратном уравнении ax² + bx + c = 0 определим коэффициенты:

$a = 1, \quad b = -5, \quad c = 6$

2. Вычисляем дискриминант

Используем формулу дискриминанта для определения количества корней:

$$D = b^2 - 4ac = (-5)^2 - 4 \cdot 1 \cdot 6 = 25 - 24 = 1$$

3. Анализируем результат

Так как дискриминант D = 1 > 0, уравнение имеет два различных действительных корня.

4. Находим первый корень

Применяем формулу корней квадратного уравнения:

$$x_1 = \frac{-b + \sqrt{D}}{2a} = \frac{5 + \sqrt{1}}{2 \cdot 1} = \frac{5 + 1}{2} = 3$$

5. Находим второй корень

Используем ту же формулу с другим знаком:

$$x_2 = \frac{-b - \sqrt{D}}{2a} = \frac{5 - \sqrt{1}}{2 \cdot 1} = \frac{5 - 1}{2} = 2$$

Ответ: $x_1 = 3, \quad x_2 = 2$
```
