4. Confidence-флаги — high / medium / low / unreadable

5. Контракт «Часть 1 сразу, Часть 2 после approval»
На экране ученика после Submit — Часть 1 видна сразу с баллом, Часть 2 — амбер «ждёт проверки». Wow-эффект достаточный?
6. Lead-gen public link — анонимный родитель
Карточка с tutor identity сверху + большая кнопка «Начать». Lead capture (имя + Telegram) перед Submit. Звучит правильно для конверсии?
7. Parent share-link с прогресс-баром по шкале ЕГЭ
