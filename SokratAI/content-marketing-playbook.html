Blog + TG Sponsorships Playbook

v1.0 · 17 мая 2026 · 12-week blog plan + Q3 TG-sponsorship strategy

Цель: привлечение репетиторов в Сократ AI (primary) + школьники/родители ЕГЭ-ОГЭ (secondary). Этот playbook работает на горизонте 3-6 месяцев и комплементарен с быстро-движущимся IG-каналом. Блог даёт authority + long-tail organic traffic. TG sponsorships — точечные high-conversion удары по B2B-аудитории.

Часть A · Блог на sokratai.ru — SEO + GEO

1. Почему блог сейчас

Три структурных причины именно сейчас:

  1. Низкая SEO-конкуренция в нише. "AI для репетитора физики", "проверка ДЗ AI", "LMS репетитор физики" — keyword'ы с <50 ежемесячных поисков каждый, но почти ни одной competitive статьи в выдаче Yandex/Google. Это blue ocean на 6-12 месяцев.
  2. GEO (Generative Engine Optimization). LLM-поиск (ChatGPT, Perplexity, Yandex GPT, AI overviews в Google) уже забирает 15-25% search-traffic в нише edtech. Структурированный educational контент с FAQ + comparison tables — это то, что LLMs reference в ответах. Раннее присутствие = долгосрочный moat.
  3. Authority signal для conversion. Когда tutor приходит на sokratai.ru из IG / TG / DM, наличие глубоких методических статей за подписью Егора Блинова (МФТИ, 7 лет опыта) — это прямой trust-сигнал, который повышает trial-to-paid конверсию.
Что блог НЕ делает

Блог НЕ daёт быстрые trial-signups в первый месяц. Yandex/Google индексируют + ранжируют новый контент 4-12 недель. Первый measurable traffic = август-сентябрь 2026. Это инвестиция в long-tail, не quick-win канал. Для quick-win — IG (Playbook #1).

6. Technical SEO checklist для sokratai.ru

Большинство пунктов — one-time setup, потом forget.

Critical (Week 1)

  • /blog route в Vite/React с SSR/SSG (используй react-helmet-async для meta-тегов)
  • Sitemap XML, обновляется автоматически при публикации
  • robots.txt с правильными rules
  • Canonical URLs на каждой статье
  • Open Graph + Twitter cards для каждой статьи
  • Schema markup: Article + Author + FAQ (через json-ld)
  • Yandex.Webmaster + Google Search Console — verification + sitemap submission
  • Page speed: Core Web Vitals в зелёной зоне (LCP <2.5s, FID <100ms, CLS <0.1)

Important (Week 2-4)

  • Internal linking: каждая новая статья ссылается на 3-5 других; перелинковка ведёт к page authority distribution
  • Breadcrumbs (с schema markup)
  • Alt-text на всех картинках с keyword variants
  • Mobile-first индексация — проверить через Google Search Console
  • HTTPS (уже есть на sokratai.ru)
  • Lazy-loading картинок (через loading="lazy")

Nice-to-have (Week 5+)

  • RSS feed (для тех кто будет subscribe)
  • Comment system (Disqus или native — больше user-generated content = SEO boost)
  • Email subscription form в footer
  • Web stories (Google AMP-эквивалент для visual content)

7. GEO — что отличает от классического SEO

GEO (Generative Engine Optimization) — оптимизация под LLM-поиск (ChatGPT, Perplexity, YandexGPT, Google AI Overviews). К концу 2026 это будет 25-40% search-traffic в edtech.

Что LLMs любят (приоритеты)

  1. Структурированные данные. FAQ блоки, comparison tables, numbered lists, definition boxes — LLMs extract эти куски и используют в ответах.
  2. Атрибуция и авторитет. Author byline с credentials (МФТИ, 7 лет опыта), даты публикации, наличие external links на проверенные источники.
  3. Конкретные числа и факты. "По данным [источник], 68% выпускников..." → LLMs prefer цитируемые статистики.
  4. Свежесть. "Updated: 2026" в schema markup критично — LLMs penalty за outdated educational content.
  5. Direct answer formatting. Первый параграф H2-секции должен прямо отвечать на implied question.

Что LLMs игнорируют / penalty

  • Filler-контент ("в современном мире...", "не секрет, что...")
  • Sales-language без подкрепления данными
  • Контент без structured data markup
  • Дублированный или semi-AI-generated контент (LLMs detect это)
GEO-specific tactical моменты
  • В каждой tutor-статье — table comparing Sokrat AI с альтернативами. LLMs очень любят tables.
  • В каждой student-статье — explicit step-by-step solution с numbered list. ChatGPT часто берёт эти шаги напрямую.
  • FAQ в конце с 5-8 явными вопросами по теме статьи. Это форматирование, которое LLMs распознают мгновенно.
  • Schema markup: FAQPage + HowTo где применимо.

8. Production model

Per-article workflow (~3.5 часа)

  1. Outline + key insights от Егора (15-20 мин аудио): главная мысль, 3-5 примеров из практики, методологические инсайты
  2. Research keyword + competitor analysis (Vladimir, 30 мин): что уже в выдаче, gap analysis
  3. Writing draft через Claude (Vladimir + Claude, 1.5-2 часа): structured outline → full draft
  4. Review + corrections от Егора (30 мин): factual accuracy, методологические нюансы
  5. Final edit + SEO setup + publish (Vladimir, 45-60 мин): visuals, schema, internal links, OG tags

2 articles/week × 3.5 часа = ~7 часов Vladimir + ~1.5 часа Егора в неделю.

Цены, если outsource

  • Контент-райтер физик-методист на freelance: 8-15 тыс ₽/статья = 16-30 тыс ₽/нед
  • Контент-агенция RU edtech: 20-35 тыс ₽/статья = 40-70 тыс ₽/нед (premium качество, но дорого для bootstrap-стадии)
  • Текущий подход (Claude + Egor выписки): ~0 ₽ внешних расходов, ~10-12 часов чистого CEO+Egor времени в неделю

Текущий подход — оптимальный пока ты в bootstrap-стадии. Рассмотри outsource когда станет >50 платящих репов и появится cash для масштабирования.

Ожидаемый ROI на ранней стадии

Пример: интеграция в medium-канале (30k subs) за 25k ₽.

  • Reach: 30k × 50% (median view rate) = 15k impressions
  • CTR: 2.5% → ~375 кликов на landing
  • Conversion landing → trial: 20% → 75 trial signups
  • Trial → paid: 30% → 22 paying tutors
  • Revenue 12 мес (ARPU 2000 ₽ × 8 мес avg lifetime): 350k ₽
  • ROI ~14x на 12-месячном горизонте
Caveat

Эти числа — оптимистичный сценарий с правильным product-channel fit. Реальные первые 2-3 интеграции дадут ROI 1:1 — 1:3 пока landing/lead-magnet/онбординг не оптимизированы. Готовься к "learning rounds" перед получением хороших unit-economics.

11. Форматы интеграций

A · Native review (highest conversion)

Что: автор канала тестирует Сократ 1-2 недели, пишет honest review со скриншотами и реальными use-cases. Не реклама — мнение коллеги.

CTR: 3-5%

Подходит: для каналов методистов и tutor-communities, где аудитория ценит честность.

Цена: +30-50% к base pricing (автор инвестирует время в тест).

B · Sponsored introduction

Что: автор представляет Сократ как "интересный инструмент, который стоит знать". Меньше глубины, больше базовое узнавание.

CTR: 2-3%

Подходит: для больших generalist каналов (Решу ЕГЭ, Школково).

Цена: standard pricing.

C · Cross-promotion / partnership

Что: взаимная реклама — мы продвигаем их канал в TG Егора, они нас в своём. No-money exchange.

CTR: 1-2% (но zero-cost)

Подходит: для каналов сопоставимого размера и аудитории. Realistically — не для нас в Q3, потому что наш канал ещё мал.

D · Webinar partnership

Что: совместный вебинар с автором канала. "Как использовать AI в подготовке к ЕГЭ" — автор приглашает свою аудиторию, Егор ведёт + показывает Сократ.

CTR (registrations): 4-8%

Подходит: для каналов с established trust + методический фокус.

Цена: revenue share или fixed 30-50k ₽.

Параметры pitch'а

  • Длина: ≤ 200 слов. Длиннее — не дочитают.
  • Specific reference к их каналу/контенту — критично. Доказывает, что не spam.
  • 3 формата с разными уровнями investment — даёт им выбор, не upfront ask.
  • Authority signal в form Егора (МФТИ, 7 лет, co-founder) — критично для serious authors.
  • Soft close "15 минут созвон" — низкое трение, легко согласиться.
Response rate ожидания

Из 10 cold-outreach pitches типично: 4-6 ответят, 2-3 согласятся на демо, 1-2 пойдут на интеграцию. То есть нужно ~15-20 outreach попыток в Q3 чтобы получить 2-3 реальные интеграции в Tier 1+2.
