## Подсказки для Q&A на питче

**Если спросят про «как AI читает рукописное решение»:**
- Multimodal LLM (сейчас — Google Gemini 1.5/2.0 Flash через Lovable AI Gateway).
- Точность 92% — измерена на пилотных задачах ЕГЭ.
- На грант от Yandex Cloud мигрируем на YandexGPT + Yandex Cloud (Yandex AI Studio) для RU compliance + доступа к топовым моделям в российской юрисдикции.

**Если спросят про unit-economics:**
- LTV ещё не posчитано (3 платящих, только 1 месяц данных) — но 0 churn = upper bound LTV высокий.
- CAC сейчас = 0 (органический рост через Егора).
- При expansion: CAC через performance marketing для tutor-сегмента — оценочно 3–5к ₽ на платящего.

**Если спросят про защиту от копирования:**
- Защита продукта — data flywheel + кодификатор ФИПИ + Сократовские промпты, наработанные на платных пилотных репетиторах. Это не воспроизводится за 1 спринт даже с большим бюджетом.

**Если спросят про экспансию за пределы физики:**
- Phase 1 — физика ЕГЭ/ОГЭ (текущий wedge). Phase 2 — математика ЕГЭ (тот же класс tutor-сегмента, технически легче чем физика). Phase 3 — остальные предметы ЕГЭ.

**Если спросят про команду:**
- Двое со-фаундеров. CEO+CPO — Владимир, vibecoder и продуктовый человек. Co-founder — Егор Блинов, эксперт-репетитор. Найм откладываем до подтверждения PMF (≥ 50 платящих репетиторов).

**Если спросят про grants кроме Yandex:**
- Sber500, Skolkovo Foundation, Phystech Acceleration — на radar. Yandex Cloud — priority #1 из-за compliance + дистрибуция через Яндекс.Учебник.
