### Ключевой инсайт (17 интервью с пользователями "Число Т")

**7 из 9 оттекших пользователей** явно запросили AI-ассистента:
> *«Хотел бы, чтобы при подготовке "онлайн репетитор" отвечал за пару минут»*

### Алгоритм прогноза балла

```typescript
interface DiagnosticAlgorithm {
  // Входные данные
  answers: {
    taskId: string;
    isCorrect: boolean;
    timeSpent: number;
    topic: string;
    difficulty: 'easy' | 'medium' | 'hard';
  }[];
  
  // Выходные данные
  result: {
    predictedScore: number; // 0-100 баллов ЕГЭ
    confidence: number; // 0-100% уверенность прогноза
    topicScores: {
      topic: string;
      score: number; // 0-100%
      recommendation: 'strong' | 'average' | 'weak';
    }[];
    recommendedTopics: string[]; // Приоритет проработки
    nextSteps: string; // Текстовая рекомендация
  };
}

// Простой алгоритм v1
function calculatePredictedScore(answers: Answer[]): number {
  const weights = { easy: 1, medium: 2, hard: 3 };
  let totalWeight = 0;
  let correctWeight = 0;
  
  for (const answer of answers) {
    const weight = weights[answer.difficulty];
    totalWeight += weight;
    if (answer.isCorrect) correctWeight += weight;
  }
  
  const percentage = correctWeight / totalWeight;
  // Маппинг на баллы ЕГЭ (упрощённый)
  return Math.round(percentage * 100);
}
```
