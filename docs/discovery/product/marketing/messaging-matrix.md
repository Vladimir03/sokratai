# Messaging Matrix — СократAI

**Дата:** 2026-03-31
**Источники:** docs 05-09 (AJTBD research), Матрица B2B×B2C, Граф работ
**Назначение:** для каждого сегмента: Job → Pain Point → Switching Trigger → Headline → CTA. Используется для лендингов, рекламы, постов.

---

## Методология

Каждый message строится по формуле AJTBD:
1. **Job** — какую работу человек пытается выполнить
2. **Pain Point** — где текущее решение ломается
3. **Switching Trigger** — момент, когда человек готов сменить решение
4. **Headline** — формулировка ценности на языке работы
5. **CTA** — призыв к действию

## Тестируемые гипотезы для лендинга

На основе твоего опыта с Яндекс Директ (декабрь 2025) и Instagram:

### Гипотеза A: «Экономия времени репетитора» (B2B)
- Целевая аудитория: репетиторы
- Headline: «ДЗ по теме урока — за 5 минут»
- Канал: Яндекс Директ (запросы «сервис для репетиторов», «автоматизация ДЗ»)

### Гипотеза B: «Ребёнок занимается каждый день» (B2C-родители)
- Целевая аудитория: родители
- Headline: «Видьте каждый день, как ребёнок готовится к ЕГЭ»
- Канал: Яндекс Директ (запросы «подготовка к ЕГЭ физика», «репетитор физика ЕГЭ»)

### Гипотеза C: «AI-помощник для подготовки» (B2C-школьники)
- Целевая аудитория: школьники 10-11 класс
- Headline: «Застрял на физике? AI подскажет путь к решению»
- Канал: Instagram (контент от брата и сестры), Яндекс Директ

> **Ключевой инсайт:** Гипотезы A и B тестируют разные Job-формулировки, не разный дизайн. Это AJTBD A/B тест.
