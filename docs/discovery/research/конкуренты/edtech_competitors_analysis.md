# Анализ конкурентов EdTech платформ для AI-репетиторства
## ЕГЭ/ОГЭ подготовка в России

**Дата исследования**: 2026-04-02
**Фокус**: B2B2C платформа для репетиторов + студентов (16-18 лет) + родителей

---

## РОССИЙСКИЕ КОНКУРЕНТЫ

### 1. Умскул (umschool.net)

**Тип**: Онлайн-школа подготовки к ЕГЭ/ОГЭ с курсами и вебинарами

#### UX/Дизайн
- **Цветовая схема**:
  - Основной акцент: **#FA7C1E (оранжевый)** — энергичный, выделяется
  - Белый фон + чёрный текст (высокий контраст)
  - Серые элементы (#CCCCCC, #E6E6E6) — нейтральные разделители
  - Красный (#FF0202) — для аватаров и критичных элементов

- **Навигация**:
  - Горизонтальное меню в шапке (Курсы / ЕГЭ / ОГЭ / Олимпиады / Средняя школа)
  - Вложенная структура предметов при наведении
  - Переключатель «Ученики ↔ Родители» (правый край)
  - Корзина покупок интегрирована

- **Мобильная версия**:
  - Адаптивный дизайн с бургер-меню
  - Иерархия контента сохраняется на мобиле
  - CTA-кнопки ("Узнать о курсе", "Хочу консультацию") размещаются на видных местах

- **Ключевая фишка**: Разделение ролей (Ученики / Родители) на высшем уровне навигации

#### Что стоит взять:
✓ Яркий акцентный цвет (оранжевый) для моментального внимания
✓ Явное разделение по типам пользователей в меню
✓ Простая иерархия: категория → предмет → курс

### 2. РЕШУ ЕГЭ (ege.sdamgia.ru)

**Тип**: Банк задач с таймерами для подготовки к ЕГЭ (минималистичный)

#### UX/Дизайн
- **История редизайна**: Полное обновление интерфейса 2019–2021
- **Ключевые улучшения**:
  - Основная информация выделяется автоматически
  - **Таймер на странице варианта** — показывает темп решения
  - 404-страница (стандартная, но не была добавлена)

- **Философия**: Минимализм, фокус на **скорости выполнения задач**
- **Целевая аудитория**: Ученики, которые прорешивают варианты под таймер

#### Что стоит взять:
✓ Таймер как критичный элемент UX (особенно для ЕГЭ/ОГЭ)
✓ Полнэкранный режим для минимизации отвлечений
✓ Автоматическая обработка ответов

### 3. Фоксфорд (foxford.ru)

**Тип**: B2B2C платформа онлайн-репетиторства + готовые курсы

#### UX/Дизайн
- **Интерактивная доска для уроков**: Репетитор → Ученик в реальном времени
  - Оба видят, что пишет репетитор
  - Возможность поправить ошибку онлайн

- **Геймификация**:
  - За решённые задачи → баллы
  - Баллы → рейтинг среди учеников
  - Баллы обмениваются на подарки (внутренняя экономика)
  - **Визуальный прогресс**: Множество задач разной сложности

- **Форматы обучения**:
  - Курсы (асинхронные)
  - Репетиторы (синхронные, гибкий график через Skype)
  - Виртуальный класс с методическим обеспечением
  - Готовые тестовые задания (на уроке и домашка)

- **Дизайн**: Красивый (по отзывам), много заданий разной сложности

#### Что стоит взять:
✓ **Интерактивная доска в реальном времени** — критична для синхронного репетиторства
✓ **Простая геймификация** (баллы → рейтинг → подарки)
✓ Комбинация асинхронных + синхронных форматов
✓ Разделение «Курсы» и «Репетиторы» как две точки входа

### 4. 100 Балльный репетитор (100points.ru)

**Тип**: Онлайн-школа ЕГЭ/ОГЭ с куратором + эксперты как преподаватели

#### UX/Дизайн
- **Позиционирование**: Премиум сегмент (цены сопоставимы с Умскулом и 99 БАЛЛОВ)
- **Отличия**:
  - Действующие эксперты ОГЭ/ЕГЭ как преподаватели
  - Регулярные обновления контента (актуальные для 2026)
  - Кураторская поддержка + репетиторы + психолог

- **Контент**: Мемы + понятные разборы сложных тем (low-stress approach)

#### Что стоит взять:
✓ **Кератор + специалисты + психолог** — трёхуровневая поддержка
✓ Мемы как инструмент вовлечения (relatable контент)
✓ Регулярные обновления (тренды ЕГЭ)

### 5. Сотка (sotkaonline.ru / 100ege.ru)

**Тип**: Бюджетная альтернатива (подписка на все предметы дешевле, чем один курс у конкурентов)

#### Бизнес-модель (важна для SokratAI)
- **Уникальное предложение**: Один price-point за всё (instead of $100/subject)
- **Целевая аудитория**: Родители и ученики с ограниченным бюджетом

#### Что стоит взять:
✓ **Ценовая стратегия**: SokratAI может предложить план по семьям (один платёж → несколько ДЗ/предметов)

## МЕЖДУНАРОДНЫЕ КОНКУРЕНТЫ

### 6. Khan Academy + Khanmigo (khanmigo.ai)

**Тип**: Бесплатные видео-уроки + AI-репетитор на базе Socratic method

#### UX/Дизайн
- **Философия**: Socratic method = **AI направляет, но не даёт ответ**
  - Задаёт вопросы → Подсказки → Ведёт к решению
  - Ученик остаётся в центре когнитивного процесса

- **Интерфейс Khanmigo**:
  - Простое чат-окно рядом с видео-уроком
  - При входе: выбор класса + предмета
  - AI представляется теплым, дружелюбным тоном

- **2026 фишки**:
  - Upload image of math/science problem → AI解説 онлайн
  - Pilots with school districts → full rollout in 2026

#### Что стоит взять:
✓ **Socratic method в UI** — не спешит давать ответ, направляет вопросом
✓ **Image upload для проблем** — студент фотографирует задачу → AI помогает
✓ **Разделение видео-контента + AI-помощи** (два окна, но связанные)
✓ **District integration** — B2B2C потенциал

---

### 7. Photomath (приложение + math.now)

**Тип**: Мобильное приложение для решения математических задач + AI-помощь

#### UX/Дизайн
- **Главное**: Camera → фото примера → пошаговое решение (Animated Steps)
- **Интерактивность**:
  - После решения: Follow-up вопросы через MathGPT
  - Custom Animated Tutorials (как задача решается шаг за шагом)
  - Textbook Solutions (готовые разборы по книгам)

- **Модель**: Нет отдельного "teacher dashboard" (все используют один интерфейс)

#### Что стоит взять:
✓ **Камера как primary input** — мобильный-первый подход
✓ **Animated steps** — визуализация решения
✓ **Follow-up questions через AI** (не просто ответ, а диалог)
✓ **Instant support 24/7** (привлекательно для родителей)

---

### 8. Quizlet (quizlet.com)

**Тип**: Цифровые флешкарточки + мобильное приложение

#### UX/Дизайн
- **Цветовая схема**:
  - **#4255FF (Dodger Blue)** — основной акцент, сигнализирует прогресс
  - **#303545 (Ebony Clay)** — тёмный грунд, серьёзность и стабильность
  - **#FFFFFF (White)** — чистота, минимализм

- **Философия дизайна**: Серьёзное + мотивирующее, глубокое обучение + волнение от открытий

- **Режимы обучения**:
  - Flashcards (вспомни ответ)
  - Learn (интерактивный режим с вопросами)
  - Test (проверка знаний)
  - Match (игра на скорость)

- **Мобильная версия**: Основная форма использования (приложение в App Store)

#### Что стоит взять:
✓ **Синий + тёмно-серый** (серьёзно-игровой баланс) — можно адаптировать
✓ **Множество режимов одного контента** (flashcards → learn → test)
✓ **Мобильный-первый** (приложение, а не веб)

---

### 9. Brilliant.org (brilliant.org)

**Тип**: Interactive learning by doing (no videos, всё через интерактивные задачи)

#### UX/Дизайн
- **Философия**: "Learn by doing", не читать + не смотреть видео
  - Диаграммы → Манипулируй данные → Переписывай код
  - Шаг за шагом → Умный feedback (ловит ошибки)

- **Геймификация** (через Rive анимации):
  - День streaks (мотивирует)
  -色-coded pathways (визуальное направление)
  - Progress tracking → Adaptive practice sets
  - Satisfying celebrations при прогрессе

- **Дизайн-идентичность**:
  - Globe-like app icon (современный)
  - Light all-caps sans serif шрифт (читаемо)
  - PIX illustration style (прямые линии, минимальные, забавные)
  - Персонаж Blorb (mascot, engaging)

- **Навигация**: Learning paths показывают направление, но с freedom of choice

#### Что стоит взять:
✓ **Interactive exercises вместо видео** — higher engagement
✓ **Rive animations** (satisfying celebrations) — мотивирует без лишнего
✓ **Day streaks + pathways** (visual progress)
✓ **Adaptive pacing** (tracks mastery → generates practice)

## СПЕЦИАЛИЗИРОВАННЫЕ ИНСТРУМЕНТЫ ДЛЯ УЧИТЕЛЕЙ

### 10. Teachworks (teachworks.com)

**Тип**: CRM + Dashboard для репетиторов (B2B инструмент)

#### UX/Дизайн Dashboard
- **Кастомизируемые виджеты**: 20+ вариантов (ты выбираешь, что видеть)
  - Lesson Status (часы за неделю + за 3 недели)
  - Upcoming Lessons (до 31 дня)
  - Lesson Profit Snapshot (revenue / wages / gross profit)
  - Monthly Statistics (активные студенты / часы / платежи)
  - Uncompleted Lessons, Overdue Tasks

- **Дизайн**: Виджеты по сетке, очень бизнес-like, минимальные отвлечения

- **Функции репетитора**:
  - Обновить availability
  - Schedule lessons
  - Manage requests
  - Track earnings

- **Интеграции**: Payments, Calendar, CRM

#### Что стоит взять:
✓ **Кастомизируемый dashboard** (только нужные метрики)
✓ **Lesson Profit Snapshot** — репетиторы видят финансы в реальном времени
✓ **Availability + Scheduling в одном месте**

### 11. MagicSchool.ai (magicschool.ai)

**Тип**: AI platform for teachers (80+ tools для планирования, оценивания, дифференциации)

#### UX/Дизайн
- **Позиционирование**: Centralized, safe AI platform specifically for schools
- **Инструменты**:
  - Lesson plan generator (тема → objectives + activities + assessment)
  - Multiple choice assessments
  - Quizzes / Tests
  - Report card comments (с strengths + areas for growth)
  - Rubric generation
  - Custom Chatbot for class (tutor-like, customizable per class)

- **Интеграции**: Google Docs, Classroom, Canvas (используй там, где уже работаешь)

- **Data**: Учителя экономят 7+ часов в неделю на подготовке

- **Security**: SOC 2 certified, FERPA/COPPA compliant

#### Что стоит взять:
✓ **80+ инструментов в одном месте** (не разбегаться)
✓ **Custom Chatbot per class** — can adapt for SokratAI tutor AI
✓ **Integration with existing tools** (не нужно менять workflow)
✓ **Time savings narrative** (репетиторам нравится)

### 12. Brisk Teaching (briskteaching.com)

**Тип**: Chrome Extension для учителей (работает внутри Google Docs, Classroom и т.д.)

#### UX/Дизайн
- **Модель**: Chrome extension (не новый app, встраивается в Google Workspace)
- **Инструменты** (30+):
  - Lesson Plan Generator (из scratch или переделай существующий)
  - Quiz generator
  - Rubric creator
  - Reading level adaptor (дифференциация)
  - Batch feedback
  - IEP / Progress reports

- **Workflow**: Open extension → Select "Create" → Search "Lesson Plan" → Type what to teach → Get structured lesson in seconds

- **Локализация**: Generate in multiple languages

- **Pricing**: Бесплатно для учителей (базовые инструменты), Brisk Next (premium) = рекомендации + batch creation + history

#### Что стоит взять:
✓ **Chrome extension** (не требует нового app'а, работает в Docs/Classroom)
✓ **30+ инструментов с простым UX** (Search + Type + Generate)
✓ **Batch feedback** (масштабируемо для репетитора с 10+ студентами)
✓ **Free для учителей** (в т.ч. репетиторов)

### Рекомендации по дизайну SokratAI

1. **Цветовая схема**:
   - Основной акцент: **оранжевый (#FA7C1E)** или **зелёный** (как у Brilliant.org green) — энергичный, выделяется
   - Белый фон, чёрный текст (контраст, как Умскул)
   - Серые элементы (#E6E6E6) для неактивных состояний

2. **Навигация по ролям** (как Умскул):
   - Top-level toggle: Репетитор ↔ Ученик ↔ Родитель
   - Или три отдельных входа? (проверить с product)

3. **Таймеры и прогресс** (как РЕШУ ЕГЭ + Quizlet):
   - Время на задачу (особенно важно для ЕГЭ)
   - Visual progress indicator (day streaks, как Brilliant)

4. **Геймификация** (как Фоксфорд + Duolingo):
   - Баллы → рейтинг → подарки (скромно, не отвлекает от учёбы)
   - Streaks (день за днём)
   - Badges за достижения (first correct answer, 100% on test и т.д.)

5. **Интерактивная доска** (как Фоксфорд):
   - Синхронный репетиторский чат (если sync-уроки добавляются)
   - Shared annotation (репетитор объясняет, ученик видит)

6. **AI-помощь в guided chat** (как Khan Academy Socratic):
   - Не давай ответ сразу
   - Направляй вопросами
   - Пошаговые hints

7. **Родительский dashboard** (как Teachworks + MagicSchool):
   - Прогресс ребёнка по предметам
   - Оценки репетитора
   - Напоминания о deadline'ах ДЗ
   - (опционально) рекомендации для ДЗ

8. **Мобильный опыт**:
   - Приложение (как Duolingo, Quizlet) или Progressive Web App
   - Bottom navigation (главное в reach)
   - Touch-friendly: 44+ px targets (no tap delays)
