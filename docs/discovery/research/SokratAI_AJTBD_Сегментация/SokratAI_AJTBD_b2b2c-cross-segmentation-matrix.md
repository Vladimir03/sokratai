# Матрица пересечений B2B × B2C: выбор wedge-связок для СократAI

**Дата:** 25 марта 2026
**Методология:** Advanced Jobs To Be Done (AJTBD)

**Рынок:** Россия и СНГ

## Предпосылка

СократAI работает по B2B2C-модели: репетитор рекомендует → родитель платит → школьник использует. Максимальная сила продукта — в точках, где работы репетитора СОВПАДАЮТ с работами родителя и школьника. Этот документ строит систематическую матрицу пересечений пяти B2B-сегментов (репетиторы) и пяти B2C-сегментов (родители+школьники), чтобы определить оптимальную точку входа на рынок.

## Шаг 1. Матрица пересечений B2C × B2B (5×5 = 25 пар)

### Методология оценки

Каждая пара оценивается по трём критериям от 1 до 5:

- **Совпадение работ (С):** насколько Core Jobs репетитора совпадают с Core Jobs родителя+школьника? (5 = работы усиливают друг друга напрямую)
- **Сила канала (К):** насколько вероятно, что ЭТОТ тип репетитора порекомендует продукт ЭТОМУ типу родителя? (5 = рекомендация естественна)
- **Экономическая совместимость (Э):** совпадают ли ценовые ожидания? Готов ли родитель платить за продукт, который использует этот репетитор? (5 = идеальное совпадение)
- **Score = С × К × Э** (макс 125)

### Детальные оценки 25 пар

#### B2B-1 (Премиальные мини-группы) × все B2C

**B2B-1 × B2C-1 (Физика ЕГЭ: финишная прямая)**
- С = 5: Репетитор хочет мгновенную автопроверку 20+ работ и guided chat → родитель хочет ежедневную контролируемую практику и максимизацию часа репетитора → школьник хочет пошаговые подсказки и быстрый фидбек. Каждая фича выполняет работу для всех трёх сторон одновременно.
- К = 5: Премиальный репетитор ЕГЭ по физике — это именно тот, к кому приходят семьи на «финишную прямую». Рекомендация инструмента = часть сервиса.
- Э = 5: Родитель платит 2100–4000 ₽/час, общий бюджет 35 000+/мес. Доплата 500–1000 ₽/мес за AI-помощника — маргинальные расходы. Репетитор готов платить 2000–5000 ₽/мес.
- **Score = 125**

**B2B-1 × B2C-2 (Профильная математика: марафон)**
- С = 3: Работы по автопроверке и контролю прогресса пересекаются, но репетитор сегмента B2B-1 ведёт физику, а не математику. Пересечение только через мультипредметников.
- К = 2: Репетитор физики мини-групп редко контактирует с семьями, ищущими математику. Разные сообщества, разные каналы привлечения.
- Э = 4: Готовность платить у обеих сторон высокая, но продукт (мат. модуль) ещё не создан.
- **Score = 24**

**B2B-1 × B2C-3 (ОГЭ: навигация на развилке)**
- С = 2: Премиальный репетитор ЕГЭ/ДВИ работает с высоким уровнем сложности. ОГЭ — другая аудитория, другие задачи, другой темп.
- К = 2: Минимальный контакт — премиальные репетиторы ЕГЭ не работают с ОГЭ-школьниками.
- Э = 2: Ценовой разрыв: ОГЭ-семьи не платят 2100–4000 ₽/час.
- **Score = 8**

**B2B-1 × B2C-4 (Премиальные тревожные родители Москвы/СПб)**
- С = 5: Репетитор генерирует отчёты для родителей → родитель хочет детальные отчёты прогресса и максимизацию ROI → школьник получает мгновенные подсказки по C-части. Работы зеркалят друг друга.
- К = 5: Премиальные родители Москвы/СПб = целевая аудитория премиального репетитора. Прямой канал, высокое доверие.
- Э = 5: Бюджет семьи 35 000+/мес, ставка репетитора 2100–4000 ₽/час. Идеальное совпадение. Родитель воспринимает AI-инструмент как часть премиального сервиса.
- **Score = 125**

**B2B-1 × B2C-5 (Бюджетные региональные отличники)**
- С = 2: Работы по автопроверке формально пересекаются, но контекст совершенно разный (мини-группы Москвы vs одинокий региональный ученик).
- К = 1: Премиальный московский репетитор не работает с бюджетными региональными учениками. Каналы не пересекаются.
- Э = 1: Ценовой разрыв непреодолим — 400–800 ₽/час (регион) vs 2100–4000 ₽/час (Москва).
- **Score = 2**

#### B2B-2 (Выгоревшие индивидуальные) × все B2C

**B2B-2 × B2C-1 (Физика ЕГЭ: финишная прямая)**
- С = 5: AI-конструктор ДЗ за 5 мин + автопроверка по фото → для школьника это ежедневная практика с мгновенным фидбеком → для родителя — видимый контроль прогресса. Guided chat разгружает репетитора и поддерживает школьника между уроками.
- К = 4: Индивидуальный репетитор имеет доверительный контакт с семьёй, но его рекомендательный вес ниже, чем у «звёздного» наставника мини-групп.
- Э = 4: Родитель платит 1000–2000 ₽/час, общий бюджет на подготовку серьёзный (финишная прямая). Репетитор готов платить 1000–2500 ₽/мес. Есть пространство для B2C-доплаты.
- **Score = 80**

**B2B-2 × B2C-2 (Профильная математика: марафон)**
- С = 3: Работы по ДЗ-автоматизации совпадают, но B2B-2 ведёт разные предметы. Для математического марафона нужен именно мат. контент.
- К = 3: Если репетитор ведёт математику — канал работает. Но сегмент B2B-2 определён шире (ОГЭ/ЕГЭ по разным предметам).
- Э = 3: Модуль математики не создан. Даже при совпадении ценовых ожиданий, продукт сейчас не может выполнить работу.
- **Score = 27**

**B2B-2 × B2C-3 (ОГЭ: навигация на развилке)**
- С = 5: Индивидуальный репетитор ведёт ОГЭ как существенную часть нагрузки. Guided chat формирует привычку ученика к AI-тьютору. Автопроверка ДЗ = родителю видно, что ребёнок занимается. Репетитор высвобождает 2–3 слота/нед.
- К = 5: Индивидуальный репетитор 9-классника — ключевой советчик семьи на развилке «колледж vs 10 класс». Доверие максимальное, контакт постоянный.
- Э = 4: Ставка 1000–2000 ₽/час совпадает с бюджетом ОГЭ-семьи. Но общий бюджет семьи ниже, чем у ЕГЭ — готовность к доплате за AI скромнее.
- **Score = 100**

**B2B-2 × B2C-4 (Премиальные тревожные родители)**
- С = 3: Работы по автопроверке есть, но премиальные родители ожидают уровень сервиса (аналитика, кастомные отчёты), который B2B-2 не предоставляет.
- К = 2: Премиальные родители обычно выбирают премиальных репетиторов (B2B-1), а не среднеценовых индивидуальных.
- Э = 2: Ставка 1000–2000 ₽/час не соответствует ожиданиям родителей, платящих 4000 ₽/час. Разные ценовые сегменты.
- **Score = 12**

**B2B-2 × B2C-5 (Бюджетные региональные отличники)**
- С = 4: Автоматизация ДЗ + guided chat = «московское качество» для регионального ученика. Репетитор разгружается, ученик получает поддержку.
- К = 4: Многие B2B-2 репетиторы — из регионов или работают онлайн с региональными учениками. Канал прямой.
- Э = 4: Ставка 1000–2000 ₽/час на нижней границе доступна для региональных семей. AI позволяет сократить частоту занятий (с 2 до 1 в неделю) при сохранении качества.

### #1. B2B-1 × B2C-1 — «Элитная физика: финишный рывок» (Score 125)

**Как работает цепочка решения:**

Триггер репетитора: «Когда у меня 15–50 учеников в мини-группах и до ЕГЭ остаётся 2 месяца, хочу мгновенно проверить 20+ работ за вечер вместо 3 часов ручной проверки, чтобы утром дать каждому персональный разбор ошибок и не выгореть к маю.»

Триггер родителя: «Когда мой ребёнок готовится к ЕГЭ по физике и я плачу 3000 ₽/час репетитору, хочу видеть, что ребёнок каждый день решает задачи и получает фидбек, чтобы быть уверенным, что деньги работают на результат.»

Точка совпадения работ: **Автопроверка ДЗ с мгновенным фидбеком.** Одна фича одновременно: высвобождает 2–3 часа репетитора (его работа), даёт родителю видимость ежедневной практики (его работа), даёт школьнику пошаговые подсказки при застревании (его работа).

**Совпадающие Core Jobs:**

| Работа репетитора | Как выглядит для родителя | Как выглядит для школьника |
|---|---|---|
| «Когда ученики отправляют фото решений, хочу получить автопроверку с выделением ошибок, чтобы за 10 мин дать обратную связь вместо часа» | «Я вижу в отчёте, что ребёнок сделал ДЗ, получил оценку 7/10, и AI разобрал ошибки — контроль без вмешательства» | «Я отправил решение, через 30 секунд увидел, где ошибся, получил подсказку — не жду репетитора до завтра» |
| «Когда ученик застрял между уроками, хочу, чтобы guided chat провёл его через рассуждение, чтобы я не тратил время на базовые вопросы» | «Ребёнок занимается даже когда репетитор недоступен — деньги работают 24/7» | «Когда застрял в 23:00, задал вопрос AI — получил наводящий вопрос, дошёл до ответа сам, чувствую уверенность» |

**Конфликты работ:**
- Репетитор хочет, чтобы AI не заменял его экспертизу («ученик должен ценить МОЙ разбор, а не AI»). Решение: AI даёт подсказки, но «экспертный разбор» подписан именем репетитора.

**Экономика связки:**
- Репетитор: 2000–5000 ₽/мес подписка B2B

- Revenue на репетитора: 2000–5000 (B2B) + 15×500 (B2C) = **9 500–30 000 ₽/мес**

**Wedge-потенциал:**
- Идеальный первый пилот. Продукт уже существует (физика, ДЗ-конструктор, guided chat, автопроверка). Фокус = физика ЕГЭ. TAM через 30–50 репетиторов → 450–2500 учеников. SOM 1-й год: 30–75 млн ₽.
- MVP уже обслуживает: конструктор ДЗ, guided chat, автопроверка по фото, отчёты. Нужно добавить: отчёты для родителей (1 клик), улучшение точности AI-проверки C-части.

### #2. B2B-1 × B2C-4 — «Премиум-контроль: снижение тревожности» (Score 125)

Точка совпадения работ: **Автоматическая аналитика и отчётность.** Репетитор экономит время на отчётах → родитель получает ROI-метрики → школьник получает мгновенные подсказки (побочный эффект системы, которая собирает данные).

- Сильный, но ВТОРИЧНЫЙ wedge после #1. Основная разница — акцент на отчётах, а не на ДЗ. Требует: дашборд для родителей, еженедельные отчёты, аналитика по темам.
- Фактически, это РАСШИРЕНИЕ #1 для premium-сегмента. Можно запускать параллельно через premium-тариф.
- Риск: начинать с этого опасно — отчёты без ДЗ-системы бесполезны.

## Шаг 3. Стратегическая рекомендация

### 1. Wedge-связка (первый пилот): B2B-1 × B2C-1 — «Элитная физика: финишный рывок»

**Почему именно эта связка:**

Максимальный Score (125) подтверждается тремя структурными преимуществами:

Во-первых, **продукт уже существует.** Конструктор ДЗ, guided chat, автопроверка по фото, база задач по физике — всё это реализовано в MVP. Не нужно строить новые модули.

Во-вторых, **экономика самоусиливающаяся.** Один репетитор (2000–5000 ₽/мес B2B) приводит 15–50 учеников (по 500–1000 ₽/мес B2C). Revenue на одного репетитора = 9 500–30 000 ₽/мес. Для достижения 30 млн ₽/год ARR достаточно 100–250 репетиторов.

В-третьих, **канал = продукт.** Премиальный репетитор рекомендует СократAI не из альтруизма, а потому что это усиливает его собственный бизнес (автопроверка, отчёты, статус «технологичного наставника»). Родитель платит не за «AI-сервис», а за «часть сервиса моего репетитора».

**Расширение #2 (B2B-1 × B2C-4)** запускается параллельно как premium-тариф того же продукта — добавляются расширенные отчёты для родителей, еженедельная аналитика.

### 2. Последовательность расширения

**Горизонт 1 (месяцы 1–3): B2B-1 × B2C-1 + B2C-4**
- Фокус: 30–50 премиальных репетиторов по физике ЕГЭ
- Работы, закрытые: автопроверка ДЗ, guided chat, отчёты для родителей
- KPI: 30 платящих репетиторов, 200+ активных учеников
- Обоснование: максимальный Score, минимальные доработки MVP, быстрая валидация unit-экономики

**Горизонт 2 (месяцы 3–6): + B2B-2 × B2C-3**
- Добавляется: массовый сегмент индивидуальных репетиторов ОГЭ/ЕГЭ
- Новые работы: упрощённый onboarding (5 мин до первого ДЗ), более мягкий guided chat для ОГЭ, управление оплатами через Telegram
- KPI: 200+ репетиторов, 2000+ учеников
- Обоснование: тот же продукт, другой ценовой сегмент. TAM в 10 раз больше. Фичи из Горизонта 1 переиспользуются полностью.

**Горизонт 3 (месяцы 6–9): + B2B-4 × B2C-5**
- Добавляется: студенты-репетиторы и региональные ученики
- Новые работы: фримиум-тариф, mobile-first UX, расширенная база задач ОГЭ, «вау-эффект» при onboarding
- KPI: 500+ репетиторов (в т.ч. 200 студентов), выход в 10+ регионов
- Обоснование: виральный сегмент (студенты делятся в чатах) + крупнейший по TAM B2C-сегмент (70% учеников РФ в регионах). Низкий ARPU компенсируется массой.

**Горизонт 4 (месяцы 9–15): + B2B-3 × B2C-2 и B2C-4**
- Добавляется: мультипредметные репетиторы, модуль профильной математики
- Новые работы: единый дашборд по двум предметам, мат. база знаний, кросс-предметная аналитика
- KPI: 1000+ репетиторов, модуль математики в production
- Обоснование: математика — крупнейший предмет ЕГЭ (336 675 участников). B2B-3 — мост между физикой и математикой. TAM математики = ₽48 млрд. Но требует создания нового контентного модуля.

### 3. Ключевые риски: где цепочка может сломаться

**Риск 1: Репетитор рекомендует, но родитель не видит ценности.**
- Точка слома: родитель воспринимает AI как «замену» репетитору и отказывается платить.
- Как проверить быстро: 10 интервью с родителями учеников текущих пилотных репетиторов. Вопрос: «Если бы ваш репетитор сказал, что использует AI-помощник для проверки ДЗ — вы бы заплатили 500 ₽/мес за доступ к отчётам?» Целевой ответ: 7 из 10 → «да».
- Стоимость проверки: 0 ₽ (телефонные звонки).

**Риск 2: Школьник не использует guided chat.**
- Точка слома: ученик игнорирует ДЗ или отвечает формально, AI-фидбек не генерирует ценности.
- Как проверить: метрика «% учеников, которые ответили на ДЗ в течение 24 часов». Порог: >60%. Если ниже — проблема в UX или мотивации.
- Стоимость проверки: 0 ₽ (аналитика в продукте).

**Риск 3: Точность AI-проверки недостаточна для C-части ЕГЭ.**
- Точка слома: AI неправильно оценивает развёрнутые решения → репетитор перестаёт доверять → отток.
- Как проверить: 50 решений C-части от реальных учеников → AI-оценка vs экспертная оценка. Порог согласованности: >85%. Если ниже — нужна доработка промптов или fine-tuning.
- Стоимость проверки: 5 000 ₽ (время эксперта на оценку 50 работ).

**Риск 4: Студенты-репетиторы (B2B-4) используют бесплатно и не конвертируются в платящих.**
- Точка слома: фримиум-модель привлекает users, но не revenue.
- Как проверить: когортная аналитика: % студентов, перешедших на платный тариф через 30 дней. Порог: >15%.
- Стоимость проверки: 0 ₽ (аналитика).

**Риск 5: Региональные семьи (B2C-5) не готовы платить даже 300 ₽/мес за AI.**
- Точка слома: ценность AI не ощущается достаточно сильно при низком бюджете.
- Как проверить: A/B тест: группа с бесплатным AI (включено в ставку репетитора) vs группа с отдельной подпиской. Метрика: retention учеников через 30 дней.
- Стоимость проверки: 0 ₽ (A/B тест в продукте).

### 4. Инсайт о ценообразовании: кто платит в каждой связке

| Связка | Кто платит | Модель | Обоснование |
|---|---|---|---|
| #1 B2B-1 × B2C-1 | **Оба** | Репетитор: 2000–5000/мес (B2B). Родитель: 500–1000/мес (B2C) | Репетитор платит за экономию времени и масштабирование. Родитель платит за видимость прогресса и 24/7 доступ к AI. Обе стороны получают разную ценность от одного продукта. |
| #2 B2B-1 × B2C-4 | **Оба (premium)** | Репетитор: 3000–5000/мес. Родитель: 1000–2000/мес | Премиальный сегмент готов платить больше за аналитику. Родитель покупает «снижение тревожности» — готовность платить выше. |
| #3 B2B-2 × B2C-3 | **Преимущественно репетитор** | Репетитор: 1000–2500/мес. Родитель: 0–500/мес (лучше включить в B2B) | ОГЭ-семьи чувствительны к цене. Выгоднее включить B2C-доступ в подписку репетитора и дать ему возможность позиционировать AI как «часть пакета». |
| #4 B2B-4 × B2C-5 | **Репетитор (фримиум → платный)** | Репетитор: 0–1000/мес. Родитель: включено | Студент должен начать бесплатно (синдром самозванца + низкий доход). Конвертация через value-wall (>5 учеников → платный тариф). Региональный родитель не платит отдельно. |
| #5 B2B-3 × B2C-4 | **Оба (premium)** | Репетитор: 3000–5000/мес. Родитель: 1000–2000/мес | Мультипредметный тариф дороже (два предмета = больше ценности). Родитель платит за единый дашборд — максимальное WTP в этой связке. |

**Стратегический вывод по ценообразованию:**

B2B-подписка репетитора — основа monetization на старте. B2C-доплата родителя — масштабирование revenue в premium-сегментах (#1, #2, #5). В бюджетных связках (#3, #4) — B2C-доступ включён в B2B-подписку.

Оптимальная тарифная линейка:
- **Старт** (фримиум): 0 ₽, до 3 учеников, базовый ДЗ-конструктор → для B2B-4
- **Базовый**: 1500 ₽/мес, до 15 учеников, ДЗ + автопроверка + guided chat → для B2B-2, B2B-5
- **Профессиональный**: 3000 ₽/мес, до 50 учеников, всё из базового + отчёты для родителей + аналитика → для B2B-1
- **Мультипредметный**: 5000 ₽/мес, 2+ предмета, единый дашборд → для B2B-3 (после запуска мат. модуля)
- **B2C-доступ родителя**: 500–2000 ₽/мес (отдельная подписка или включено в тариф репетитора)
