# Mini App Solution Parsing - Implementation

## Что было сделано

Исправлена проблема "Решение не найдено" в Mini App и реализовано правильное отображение решений с LaTeX формулами.

## Новые функции в `supabase/functions/telegram-bot/index.ts`

### 1. `extractLatexFormulas(text: string)` 
**Строки: 977-1011**

Извлекает LaTeX формулы из текста:
- Display mode: `$$...$$` 
- Inline mode: `$...$` (только если содержат LaTeX команды)
- Возвращает массив формул и текст без них

```typescript
{ formulas: string[], textWithoutFormulas: string }
```

### 2. `extractFinalAnswer(aiResponse: string)`
**Строки: 1013-1034**

Извлекает финальный ответ из AI ответа:
- Паттерны: "**Ответ:**", "**Итог:**", "**Финальный ответ:**"
- Возвращает `string | null`

### 3. `parseSolutionSteps(aiResponse: string)` - Переписана
**Строки: 1036-1133**

Новая логика парсинга:
1. **Находит секции** через различные паттерны заголовков:
   - `### Заголовок`
   - `**Заголовок:**`
   - `1. Заголовок`
   - `**Шаг N:**`

2. **Извлекает для каждого шага:**
   - `number` - номер шага
   - `title` - заголовок
   - `content` - основной текст (без формул)
   - `formula` - главная LaTeX формула
   - `method` - подсказка о методе (если есть)

3. **Fallback механизмы:**
   - Если структурированные секции не найдены → ищет простую нумерацию
   - Если и это не сработало → создает один шаг со всем текстом

### 4. `saveSolution()` - Обновлена
**Строки: 1135-1184**

Изменения:
- Парсит RAW ответ от AI (до форматирования для Telegram)
- Использует `extractFinalAnswer()` для получения ответа
- Логирует количество шагов и финальный ответ
- Сохраняет правильную структуру в БД

## Пример трансформации

### Исходный ответ от AI:
```
Хорошо, давай решим это квадратное уравнение.

### Шаг 1: Определяем коэффициенты

У нас есть уравнение: $$x^2 + 5x - 10 = 0$$

Коэффициенты: $a = 1$, $b = 5$, $c = -10$

### Шаг 2: Вычисляем дискриминант

Формула дискриминанта: $$D = b^2 - 4ac$$

Подставляем: $$D = 5^2 - 4 \cdot 1 \cdot (-10) = 25 + 40 = 65$$

**Ответ:** $x_{1,2} = \frac{-5 \pm \sqrt{65}}{2}$
```

## Отображение в Mini App

**Файл:** `src/pages/MiniAppSolution.tsx`

Mini App теперь правильно получает данные:
- `solution.steps` - массив шагов с `title`, `content`, `formula`
- `solution.finalAnswer` - финальный ответ с LaTeX
- LaTeX рендерится через KaTeX в компонентах `Math` и `MathBlock`
