<a id="layer-1"></a>
# Уровень 1 — GTM на одном дыхании

## One-liner (используется везде)
**Сократ AI — рабочее место репетитора: проверяет рукописные домашки по критериям экзамена, ведёт ученика подсказками без готовых ответов и собирает ДЗ из вашей базы за минуты.**

## Единое сообщение, через которое идёт всё
Репетитор должен поверить в одно: **«Я могу вести больше учеников, не проверяя ДЗ ночами и не теряя качество — потому что AI делает черновую проверку, а финальное слово остаётся за мной».** Это «второй взгляд, а не замена» — формула, проверенная кастдевами. [почему именно это ▸](#l2-message)

## Для кого
Экзаменационные репетиторы с 10+ учениками или мини-группами, упёршиеся в потолок: ручная проверка стопок фото решений съедает вечера, расписание забито, доход больше не растёт. [кто покупатель ▸](#l2-buyer)

## Что проверить до масштабирования
Тезис «сможете вести больше учеников» пока не подтверждён ни одним именованным кейсом с числами — есть только «8 часов в неделю» Егора. Собрать 1–2 кейса с числом учеников ДО/ПОСЛЕ — до масштабирования оффера про ёмкость. [что тестировать ▸](#l2-test)

> **Валидационный долг:** пак стоит на **3** непроверенных допущениях, **1** фатальное: (а) что «больше учеников» — реальный результат использования (фатально для оффера V1), (б) что оффер через доход конвертит лучше оффера через страховку, (в) что холодный посетитель понимает категорию «AI проверяет рукописное» без демонстрации. [риски входных данных ▸](#l2-input-risks)

> ⚠️ Ценность продукта подтверждена продажами и кастдевами (3 платящих, «второй взгляд» и проверка рукописного = реальные Aha-моменты). НЕ подтверждена ось «доход/ёмкость» — офферы V1/V2 тестовые, не факты для масштабирования.

# Уровень 2 — почему копирайт такой

<a id="l2-input-risks"></a>
## Что вы дали — и риски, которые я в этом вижу

| Вход | Как использовал | Риск | Быстрая проверка |
|---|---|---|---|
| «Увеличить доход/учеников в 2 раза» (позиция владельца из переписки с Сашей) | Гипотеза для V1, смягчена до «вести в 2 раза больше учеников» | Ни один кейс не подтверждает «×2»; продукт не приводит лидов — обещать рост дохода целиком = обещать больше, чем выполняем → разочарование | Спросить Егора/Елену: число учеников до/после, часы до/после |
| «8 часов в неделю» (Егор, свой канал) | Пруф-цифра V2 и блока ценности | Один человек, сооснователь — предвзятый источник для холодного посетителя | Подтвердить у 1–2 внешних платящих; иначе подпись «сооснователь» |
| «Второй взгляд поймал ошибку» (Елена, кастдев) | Ядро V3 + блок страхов | Слабых рисков нет — дословная цитата платящего; следить только за анонимизацией | Разрешение Елены на публикацию истории |
| Кастдев-интел «чего не говорить» | Жёсткий фильтр всех блоков | Выборка 3 человека — «не давить на усталость» может не обобщаться на массовый сегмент | 5-секундные тесты в чате сообщества |

<a id="l2-message"></a>
## Почему именно это сообщение
Мотивация живёт уровнем выше ежедневных задач: репетитор покупает не «проверку ДЗ», а свою большую цель — **доход и жизнь без ночной рутины при сохранении статуса сильного преподавателя** (Big Job). Но категория «AI проверяет рукописные работы по критериям» для большинства **незнакома** (кастдев: «ученики не знали, что от руки можно проверять») — значит, лендинг обязан сначала показать, что такой способ вообще существует, и лишь затем сравнивать. Главный страх, убивающий покупку, — «AI заменит меня / испортит мою репутацию ошибкой» — снимается рамкой «второй взгляд, финальное слово за вами», и она же отстраивает от ChatGPT (там ученик списывает, у нас — не может).

<a id="l2-buyer"></a>
## Кто покупатель и что уже доказано
Два целевых сегмента: премиальные наставники мини-групп (15–50 учеников, чек 2100–4000 ₽/час) и высоконагруженные индивидуалы на пороге выгорания (10–20 учеников, 1000–2000 ₽/час). Ценность подтверждена: 3 платящих, задокументированные Aha-моменты (проверка рукописного, «второй взгляд», Часть 2 по критериям). Не доказано: массовая конверсия холодного трафика и ось «ёмкость→доход».

## Логика лендинга простыми словами
Порядок блоков = путь недоверчивого посетителя: узнал свою ситуацию → увидел конкретную выгоду в цифрах → увидел доказательство (как работает) → потрогал (вау-демонстрация) → понял, чем плох текущий способ → страхи сняты → представил жизнь «после» → сделал дешёвый первый шаг (триал без карты). Это совпадает со структурой, которую рекомендовал Саша (ценность → оффер+вау → функционал → соцдоказательства → регистрация) — методология лишь добавляет блоки «чем плох текущий способ» и «страхи».

<a id="l2-test"></a>
## Что тестировать первым
1. Оффер: V1 (ёмкость/доход) против V3 (второй взгляд) — 5-секундный тест в чате сообщества, потом последовательный замер конверсии по неделям (A/B невозможен на текущем трафике).
2. Цифры для пруфов: ученики/часы до-после у Егора, Елены, Эмилии — без них V1 остаётся с тегами [VERIFY].

<a id="layer-3"></a>
# Уровень 3 — полный пак (Part 1: лендинг)

## 0. One-liner
**Сократ AI — рабочее место репетитора: проверяет рукописные домашки по критериям экзамена, ведёт ученика подсказками без готовых ответов и собирает ДЗ из вашей базы за минуты.**
Тест: незнакомец должен повторить «что это» (рабочее место репетитора), «что делает» (проверяет рукописное ДЗ, ведёт ученика, собирает ДЗ) и «чем лучше» (по критериям, без готовых ответов, за минуты).

<a id="l3-landing"></a>
## Part 1 — Варианты hero-оффера (выбор на GATE A)

### V1 — Ёмкость → доход (ось владельца, честная формулировка)
> **H1:** Ведите в 2 раза больше учеников — без ночной проверки ДЗ [VERIFY — нет кейса «×2»; безопасная версия без числа: «Берите больше учеников — без ночной проверки ДЗ»]
> **Подзаголовок:** Сократ проверяет рукописные домашки по критериям ФИПИ и ведёт ученика к решению, пока вас нет рядом. Вы только подтверждаете балл — за минуты, а не за вечер.
> **CTA:** Попробовать 7 дней бесплатно — без карты
> **Пруф-строка:** Егор, преподаватель МФТИ: освободил ~8 часов в неделю [VERIFY — подтвердить цифру у Егора для публичного использования]

*Что делает:* бьёт в большую цель (доход/масштаб), механику подставляет как доказательство. Риск: «в 2 раза» без кейса = завышенное обещание; «больше учеников» требует, чтобы читатель сам достроил «→ больше денег» (обычно достраивает).

### V2 — Время с цифрой (пруф Егора)
> **H1:** Верните себе 8 часов в неделю — домашки проверит Сократ [VERIFY — цифра Егора]
> **Подзаголовок:** Рукописные решения, Часть 2 ЕГЭ по критериям ФИПИ, подсказки ученику без готовых ответов. Финальное слово — всегда за вами.
> **CTA:** Попробовать 7 дней бесплатно — без карты

*Что делает:* самая конкретная цифра из всех, мгновенно считывается. Риск: у tough-профи месседж «снимаем нагрузку» вызывает «я и так справляюсь» (кастдев Елены) — но здесь через результат-часы, не через «усталость», это смягчает.

### V3 — Второй взгляд (проверено кастдевом, страховка от ошибки)
> **H1:** AI-проверка домашек, которая ловит даже то, что пропустили вы
> **Подзаголовок:** «Я проверила задачу — вроде всё хорошо. А Сократ ставит 1 балл из 2 — и оказался прав». Рукописные работы, критерии ФИПИ, балл подтверждаете вы. [VERIFY — согласие Елены на цитату]
> **CTA:** Проверить первую работу бесплатно

*Что делает:* дословный Aha-момент платящего клиента прямо в hero; снимает страх замены самой формулировкой. Риск: уже по охвату — активирует «страховку качества», а не «рост дохода»; сильнее для опытных профи, слабее для студентов-новичков.

### V4 — AI-ассистент вместо куратора (рефрейм категории, ценовой якорь)
> **H1:** AI-ассистент репетитора: проверяет домашки и ведёт учеников между занятиями
> **Подзаголовок:** Как нанять куратора — только за 200 ₽ в месяц, а не за 30 000 [VERIFY — рыночная ставка куратора]. Проверка рукописного по критериям, подсказки ученикам, учёт оплат — в одном месте.
> **CTA:** Нанять AI-ассистента на 7 дней бесплатно

*Что делает:* будит категорию «делегируй сотруднику» (идея методайзеров «нейронка делает работу за репетитора») + мощный ценовой якорь. Риск: «нанять ассистента» может прозвучать как «заменить себя» — ровно тот страх, который снимаем; требует немедленной оговорки «финальное слово за вами».

**Рекомендация скилла:** лид — V1 (безопасная версия без «×2», пока нет кейса) ИЛИ V3 для tough-профи-трафика; V2 — как пруф-строка внутри любого варианта, не как H1; V4 — тест в рекламе, не на лендинге (нужна секунда на «не замена»).

## Part 1 — Полная последовательность лендинга (готовый текст)

### Блок 1. Hero
H1/подзаголовок/CTA — выбранный вариант выше. Под CTA чипы: «Без карты · Полный доступ · Потом 200 ₽ первый месяц · Отмена в один клик». Регалии основателей ИЗ hero убрать (переезжают в блок 7).

### Блок 2. Для чего вы его нанимаете (фокус-работы)
**Заголовок:** Сократ берёт на себя три самые дорогие рутины
- **Проверить** — рукописные решения с фото, по критериям ФИПИ/CEFR, с черновиком фидбека каждому ученику.
- **Довести** — ученик застрял в 23:00? Сократ ведёт его наводящими вопросами до ответа. Готовое решение не выдаёт — списать не получится.
- **Собрать** — ДЗ из вашей базы или банка задач за 5 минут, выдача в один клик [VERIFY — «5 минут» из wedge-памятки, подтверждён пилотом].

### Блок 3. Узнаёте вечер? (контекст и триггер)
Двадцать фото решений пришли за три часа до группового занятия. Каждому нужен разбор — с учётом именно его ошибок. Раньше это был ваш вечер. Теперь это 10 минут подтверждения того, что Сократ уже проверил.

### Блок 4. Ценность в цифрах (не прилагательные)
- Проверка рукописной работы: **минуты вместо 20–30 минут ручного разбора** [VERIFY — замер на реальных работах].
- Часть 2 ЕГЭ — **по критериям ФИПИ**, покритериальный разбор, как на экзамене.
- ~**8 часов в неделю** обратно — проверка ДЗ + учёт оплат + отчёты родителям [VERIFY — Егор].
- Ученик получает подсказку **в ту же минуту**, а не «когда вы освободитесь».

### Блок 5. Как это работает (фичи как доказательство)
1. Выдаёте ДЗ из своей базы (или грузите PDF — Сократ распознает до 73 задач разом [VERIFY — внутренний тест, подтвердить формулировку]).
2. Ученик решает на телефоне, фотографирует рукописное решение.
3. Сократ проверяет по критериям и готовит разбор; застрявшего ведёт вопросами.
4. Вы подтверждаете баллы за минуты — родитель видит прогресс.

### Блок 6. Попробуйте прямо здесь (вкус Aha-момента — «вау» Саши)
Место для вау-элемента: скрин-стори «поймал ошибку» / 30-сек видео / интерактив — решение по рисерчу A4 (см. research-and-offers.md). Требование канона: первый Aha — как можно левее, до регистрации.

### Блок 7. Почему не по-старому (расстрел текущего способа)
Мессенджер + «Решу ЕГЭ» + ChatGPT — это: непдписанные фото в десяти ветках, ночная проверка, «кто быстрее — того и тапки», и ученик, который просто списывает у ChatGPT. Каждый инструмент бесплатный — а вечер всё равно ваш. *(Здесь же — блок основателей: «сделали репетиторы для репетиторов» — Егор Блинов, МФТИ, дважды 100-балльник; Владимир Камчаткин, МФТИ, Фоксфорд, Т-Образование — как пруф «понимаем вашу работу», не как hero-текст.)*

### Блок 8. Честные ответы на страхи
- **«AI заменит меня»** — нет: Сократ готовит черновик, балл и финальное слово — ваши. Ученик видит ВАШЕ имя и фото, а не робота.
- **«AI ошибётся, а пострадает моя репутация»** — каждый балл подтверждаете вы; AI — второй взгляд, который страхует, а не решает.
- **«Ученики разучатся думать»** — Сократ не выдаёт готовых ответов — ведёт вопросами (в отличие от ChatGPT).
- **«Ещё одна платформа, некогда разбираться»** — добавить ученика за секунду; первое ДЗ — за 5 минут; 7 дней без карты, отмена в один клик.

### Блок 9. Жизнь «после» (State B)
Вечер после занятий — ваш. Утром — не стопка фото, а готовые разборы на подтверждение. Родители видят прогресс без напоминаний. В расписании — на два слота больше, и это новые ученики, а не новые ночи.

### Блок 10. Первый шаг + CTA
**Попробовать 7 дней бесплатно** — без карты. Дальше 200 ₽ первый месяц, потом от 1000 ₽/мес. Кнопка на каждом экране.

> Последовательность — диагностика, не украшение: если мотивированный трафик не конвертится — ищите, какой переход сломался (не узнал ситуацию / ценность не конкретна / пруф не связан с критерием / страх остался жив / конкурент не расстрелян / CTA дороже первого Aha).
>
> <sub>**▸ methodology trace.** Последовательность блоков = короткая Критическая цепочка работ до состояния «достаточно веры для первого шага» (`communication.md §8`); ценность в критериях — §3; фичи-как-доказательство — §4; вкус Aha как можно левее (`customers-attention-management.md §6`); блоки 7–8 = компоненты 4–5 активаторов выбора (`consideration-activators.md §1`); страх «заменит меня» vs реальный блокер (`barrier-removal.md`); выбор «вести через Big Job + будить категорию» — Class 2 (`behaviour-change.md §10`, кастдев: категория незнакома).</sub>

## Приложение — что несёт каждый блок

| Блок | Компонент активации выбора |
|---|---|
| Hero (V1–V4) | 1 новый способ существует + 2 дельта ценности |
| Фокус-работы, Как работает | 3 названный продукт + первый шаг; фичи = пруф |
| Узнаёте вечер? | контекст+триггер (узнавание «это про меня») |
| Ценность в цифрах | 2 дельта в критериях сегмента |
| Вау-блок | экспериенциальная загрузка (вкус Aha) |
| Почему не по-старому | 5 расстрел конкурирующего способа |
| Честные ответы | 4 снятие специфических страхов |
| Жизнь «после» | State B / Big Job |
| CTA | 3 первый шаг с минимальной ценой (без карты) |
