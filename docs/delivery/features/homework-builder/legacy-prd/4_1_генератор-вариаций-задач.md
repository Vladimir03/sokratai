## 4.1 Генератор вариаций задач
**Цель**: генерировать 5 вариаций задачи, менять числа/условия, сохранять метод решения, автоматически проверять корректность ответа.

### In‑scope
- Edge Function: `supabase/functions/generate-variations/index.ts`
- `POST { task_text, subject, count:5, difficulty_shift:0 }`
- Ответ: `[{ task_text, correct_answer, difficulty }]`

### Валидация
- 2‑й вызов LLM: «Реши задачу» → если ответ не совпал с `correct_answer`, помечаем `needs_review=true` или возвращаем предупреждение.

### Приёмка
- Генерируются 5 уникальных вариаций.
- Большинство проходят self‑check.

### Инструкция для Codex/Claude
Реализуй функцию генерации + self‑check, лимиты токенов, защиту от дубликатов (сравнение нормализованного текста).
