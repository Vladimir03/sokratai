## L3 Результаты ДЗ (/tutor/homework/:id)
**Цель**: детальный экран результатов.

### UI
- AI Summary (градиентная карточка)
- 4 метрики (сдали/назначено, средний балл, % без ошибок, top error type)
- Таблица учеников: имя, статус, балл, ошибки
- Expand row: фото ответа, AI confidence, toggle верно/неверно, комментарий tutor
- Bar chart по задачам (ошибки/средний балл)

### API
- `GET /assignments/:id` + `GET /assignments/:id/results`
- `POST /submissions/:id/review`

### Приёмка
- Репетитор может поправить оценку и оставить комментарий.
- Данные обновляются (optimistic или refetch).
