## 4.2 Аналитика
**Цель**: дать агрегированную аналитику по домашкам и ученикам.

### In‑scope
Edge Function: `supabase/functions/homework-analytics/index.ts`

Эндпоинты:
- `GET /analytics/assignment/:id` → `summary, per_task, per_student, ai_summary`
- `GET /analytics/student/:id` → `avg_score, trend, weak_topics, strong_topics`
- `GET /analytics/tutor/overview` → `active_count, pending_review, students_needing_help`

### Метрики
- avg_score, median_score
- top_error_types
- hardest_tasks (по доле ошибок)

### Приёмка
- По assignment выдаётся корректная сводка.
- По student — тренд и слабые темы (по topic/subject/task tags если есть).

### Инструкция для Codex/Claude
Реализуй агрегирующие запросы (SQL внутри функции или view), обеспечь RLS‑совместимость (tutor видит только своё).
