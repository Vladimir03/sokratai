Каскад: avatar_url → SVG-силуэт по gender → инициалы имени на bg-accent → анонимная заглушка. Используется одинаково в Navigation, на /tutor/profile, в guided chat и в будущем в GuidedThreadViewer (Phase 5).

gender = 'male'

S3

Wedge-эффект: голос репетитора в guided chat ученика

spec §6 AC-4 · GuidedChatMessage.tsx · сейчас vs. после
Главный продуктовый эффект фичи — ученик мгновенно отличает «своего Вадима» от голосов AI. Сравнение слева (как сейчас, обезличенный label «РЕПЕТИТОР») и справа (Telegram-style identity: аватар 32×32 + имя над пузырём).

S4

Секция «Безопасность» × 3 способа входа

spec §6 · добавлено 2026-04-15 (Google OAuth в scope) · AC-11/12/13
Контракт: секция «Безопасность» рендерится в одном из 3 состояний на основе auth.identities у пользователя. A — классика (email + password), B — пришёл через «Войти через Google» (пароля нет, но есть soft-CTA «Установить пароль»), C — оба способа привязаны (mixed). Email read-only для всех вариантов где есть Google identity (email-источник истины — Google).
