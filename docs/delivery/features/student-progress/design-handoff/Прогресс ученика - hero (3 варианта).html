/tutor/students/:id · 3 варианта layout-а — сравните и выберите
a

Список-карточки по работам

Каждая работа — строка-карточка с мини-картой задач и статусом проверки. Самый прямой ответ на боль Елены: сверху вниз видно «что сдано / что проверено / что висит», клик раскрывает задачи. Подтверждение — на каждой карточке.
b

Таблица-heatmap «работы × задачи»

Плотная сетка как в Results v2: строка — работа, столбцы — задачи, цвет клетки = доля от макс. балла. За один взгляд видно провальные темы по всем работам. Sticky-колонка с названием, проверка — в правой части строки.
c

Timeline по датам с динамикой балла

График среднего балла сверху + лента работ по датам. Акцент на тренд: видно, как ученик движется во времени — удобно для разговора с родителем и для понимания спадов. Проверка — прямо в карточке даты.
