# Шкалы баллов для фичи «Прогресс ученика»
### Референс конвертаций · ЕГЭ / ОГЭ / школа · v1 (2026)

> Источник истины для `score_kind`, конвертаций и `EGE_PHYS.anchors` в макете/спеке.
> **Каверзный инвариант:** официальная шкала ФИПИ публикуется **весной**, после досрочного этапа → версионировать по году (`scale_year`). Перед прод-релизом сверять с obrnadzor.gov.ru.

## 1. ЕГЭ физика 2026 — первичный → тестовый (полная таблица)

Макс. первичный = **45** → 100 тестовых. Таблица **полная** (0–45), интерполяция не нужна — это lookup.

Пороги: **36 тест-балла** (8 первичных) = аттестат · **39** (9 первичных) = вуз Минобрнауки.

```js
// ЕГЭ физика, scale_year: 2026. Полный lookup primary -> scaled.
const EGE_PHYS_2026 = {
  scale_year: 2026, max_primary: 45,
  thresholds: { attestat_scaled: 36, vuz_scaled: 39 },
  // index = первичный балл, значение = тестовый
  map: [0,5,9,14,18,23,27,32,36,39,40,43,44,46,48,49,51,53,54,56,58,
        59,61,62,64,65,67,68,70,71,73,74,76,77,79,80,82,84,86,88,90,
        92,94,96,98,100]
};
// напр. 21 первичный -> EGE_PHYS_2026.map[21] = 59;  23 -> 62
```

(Если в макете нужны именно `anchors` для интерполяции — возьми любые точки из `map`; но лучше залить весь `map`, он точный.)

## 2. ЕГЭ физика — баллы по заданиям (для max_score задач)

Полезно и для пробника, и для ДЗ (задачные баллы):

| Задания | Первичных за задание |
|---|---|
| 1–4, 7, 8, 11–13, 16, 19, 20 | 1 |
| 5, 6, 9, 10, 14, 15, 17, 18 | 2 |
| 22, 23 | 2 |
| 21, 24, 25 | 3 |
| 26 | 4 |

Часть 1 = 28 первичных (зад. 1–20) · Часть 2 = 17 (зад. 21–26) · **ИТОГО 45**.

## 3. ЕГЭ → школьная оценка (НЕофициально, опц.)

Официального перевода ЕГЭ→оценка нет. Примерно (по тест-баллу): `5` ≥ 80 · `4` 60–79 · `3` 36–59 · `2` ≤ 35. Использовать только как справочную подпись, не как источник истины.

## 4. ОГЭ — отдельная история (per-subject)

ОГЭ-конвертация **зависит от предмета** (математика ≠ физика ≠ русский) и считается первичный→оценка (2–5) по порогам, не в 100-балльную. Пример из макета: ОГЭ-математика, макс. первичный 31, «22/31 · оценка 4». **Точные пороги по предмету — добить отдельно** перед прод-релизом ОГЭ-трека (для пилота-физики-ЕГЭ не блокер). Не угадывать — тянуть из obrnadzor.gov.ru / демоверсии ФИПИ по конкретному предмету.

## 5. Школьная оценка (трек «улучшение оценок»)

Конвертации **нет** — это **ручной ввод** `school_grade` (2–5) репетитором (`score_kind: 'school_grade'`). Цель ученика — тоже оценка (3 → 4). Никакой таблицы не нужно.

## Контракт для спеки
- `score_kind`: `primary` | `ege_scaled` | `oge_grade` | `school_grade`.
- `scale_year` на работе/пробнике (таблицы меняются ежегодно).
- ЕГЭ-100 и ОГЭ-оценка — **производные** от первичных через версионируемую таблицу; первичные баллы = хранимая истина.
- Цвет ячеек везде = % от max (универсально, exam-agnostic).
