# Senior Analyst Studio · Сократ AI — стартовые аналитические артефакты

Папка содержит стартовые материалы для двух 10-недельных проектов менторской программы Senior
Analyst Studio. Оба проекта идут на **реальных, но обезличенных данных Сократа под NDA**.
Артефакты заземлены на то, что в продукте реально есть (проверено по коду и миграциям 2026-06-22),
а не на общих предположениях.

## Жёсткие ограничения (для обоих проектов)

- **Несовершеннолетние ученики** → работаем только на обезличенных данных под NDA. Нигде не выводим ПДн.
- **AI-проверка ДЗ работает на Gemini** (через Lovable AI Gateway). **Агент отчётов задуман на Claude.**
- **Не выдумываем данные и метрики, которых нет.** Чего нет — помечаем «собрать/подтвердить».

## Легенда пометок (общая для всех файлов)

- **✓ РЕАЛЬНО** — есть в коде/схеме Сократа.
- **⚠ ПОДТВЕРДИТЬ** — нужно проверить на данных или у владельца.
- **✚ СОБРАТЬ** — отсутствует, нужно построить.

## ⚠ Важное предупреждение про данные (не гоняться за призраками)

Таблицы `homework_tutor_submissions` / `homework_tutor_submission_items` (с колонками
`recognized_text`, `ai_feedback`, `ai_error_type`, `student_image_urls`) и колонка `workflow_mode`
**удалены** миграцией `20260406120000_drop_classic_homework.sql` (classic-режим дропнут). Их в
проектах использовать нельзя.

**Следствие:** OCR-текста рукописи в Сократе **не существует**. Фото решения уходит в Gemini
мультимодально в реальном времени; распознанный текст нигде не сохраняется. Это меняет дизайн
метрики «точность распознавания» (Артефакт 1) и фичи «грязная рукопись» (Артефакт 3).
