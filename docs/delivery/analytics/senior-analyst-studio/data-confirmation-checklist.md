# Данные для подтверждения — Сократ × Senior Analyst Studio

Единый чек-лист для **владельца** (и студенток) перед стартом недели 2. Сюда сведены все открытые
вопросы из трёх артефактов, без дублей, сгруппированы по приоритету. Помечено, какой проект блокирует
каждый пункт: **[Д]** — Катя-Дубай (качество проверки), **[К]** — Катя-Кипр (успеваемость/отчёты),
**[оба]**.

**Как пользоваться:** пройди по `- [ ]`, впиши ответ после `→ ОТВЕТ:`. Пункты с пометкой
**РЕШЕНИЕ ВЛАДЕЛЬЦА** — это не «данные», а выбор, который никто кроме тебя не сделает.

## Приложение — SQL-заготовки (self-serve для владельца)

⚠ Приблизительно: **сверь имена колонок со схемой перед запуском** (составлено по коду, не прогнано
на БД). Закрывают большинство P0-объёмов.

```sql
-- 1) Масштаб проверок (всего и за 30 дней)
select
  count(*)                                                   as checks_total,
  count(*) filter (where ai_score is not null)               as ai_graded,
  count(*) filter (where created_at > now() - interval '30 day') as last_30d
from homework_tutor_task_states;

-- 2) Насколько репетиторы правят AI (потолок лейблов F1–F3)
select
  count(*) filter (where ai_score is not null)            as ai_checks,
  count(*) filter (where tutor_score_override is not null) as overridden,
  count(*) filter (where tutor_reviewed_at is not null)    as reviewed,
  count(*) filter (where tutor_force_completed_at is not null) as force_completed
from homework_tutor_task_states;

-- 3) Состав задач: формат проверки × тип ответа
select check_format, task_kind, count(*) 
from homework_tutor_tasks
group by 1, 2 order by 3 desc;

-- 4) Покрытие эталоном (reference для D1)
select
  count(*)                                                              as tasks,
  count(*) filter (where solution_text is not null
                     and length(trim(solution_text)) > 0)              as with_solution
from homework_tutor_tasks;

-- 5) Цель заполнена + трек (для метки «отстал» вариантов A/C)
select
  count(*)                                       as students,
  count(*) filter (where target_score is not null) as with_target,
  count(*) filter (where exam_type = 'ege')        as ege,
  count(*) filter (where exam_type = 'oge')        as oge
from tutor_students
where archived_at is null;

-- 6) Подтверждённые пробники на ученика (жизнеспособность варианта A)
select
  count(distinct student_id) filter (
    where status in ('approved', 'manually_entered')
  ) as students_with_confirmed_mock
from mock_exam_attempts;
```

Доля «с фото» (`submission_payload.photos`) и язык vs физика (через `homework_tutor_assignments.subject`)
считаются отдельно — напишу точные запросы, как подтвердишь доступ и имена колонок.
