# Telegram Bot Formatting Improvements

## Summary

Улучшено форматирование сообщений от ИИ в Telegram боте для повышения читаемости и приближения к веб-версии чата.

## Изменения в файле `supabase/functions/telegram-bot/index.ts`

### 1. Новая функция `convertMarkdownHeadings()`
**Строки: 672-685**

Преобразует markdown заголовки в жирный текст с пустыми строками:
- `# Заголовок` → `\n**Заголовок**\n`
- `## Заголовок` → `\n**Заголовок**\n`
- `### Заголовок` → `\n**Заголовок**\n`

### 2. Новая функция `convertMarkdownLists()`
**Строки: 687-748**

Преобразует markdown списки в emoji-маркеры:
- Нумерованные списки: `1. Текст` → `1️⃣ Текст` (до 10 пунктов)
- Буллетные списки: `- Текст` или `* Текст` → `📌 Текст`

### 3. Новая функция `addBlockSpacing()`
**Строки: 750-786**

Автоматически добавляет пустые строки между:
- Заголовками и контентом
- Списками и параграфами
- Различными блоками контента
- Очищает излишние переносы строк (более 2 подряд)

### 4. Новая функция `isComplexFormula()`
**Строки: 788-805**

Определяет сложность LaTeX формулы по критериям:
- Длина > 50 символов
- Несколько вложенных `\frac`
- Наличие матриц, интегралов, сумм
- Более 3 вложенных фигурных скобок

### 5. Улучшена функция `preprocessLatex()`
**Строки: 615-688**

Добавлено:
- Обнаружение сложных формул перед обработкой
- Автоматическая вставка подсказки о Mini App для сложных формул:
  ```
  📱 <i>Для красивого отображения формул открой Mini App ниже</i>
  ```

### 6. Улучшена функция `convertMarkdownToTelegramHTML()`
**Строки: 807-822**

Добавлено:
- Поддержка блоков кода: \`\`\`код\`\`\` → `<pre>код</pre>`

### 7. Обновлена функция `formatForTelegram()`
**Строки: 824-849**

Изменен порядок применения трансформаций:
1. Преобразование заголовков (convertMarkdownHeadings)
2. Преобразование списков (convertMarkdownLists)
3. Предобработка LaTeX (preprocessLatex)
4. Конвертация LaTeX в Unicode (convertLatexToUnicode)
5. Добавление отступов (addBlockSpacing)
6. Конвертация markdown в HTML (convertMarkdownToTelegramHTML)

**После:**
```
**Хорошо, давай разберем шаг "Вычислить дискриминант"**

Что такое дискриминант?

Дискриминант (обозначается как D) — это специальное число.

**Формула дискриминанта:**
D = b² - 4ac
```

## Преимущества

1. **Улучшенная читаемость**: Четкая визуальная иерархия с заголовками и отступами
2. **Структурированность**: Списки с emoji-маркерами легче воспринимаются
3. **Удобство**: Автоматическая подсказка о Mini App для сложных формул
4. **Консистентность**: Форматирование приближено к веб-версии чата

## Ограничения Telegram учтены

- Используются только поддерживаемые HTML теги: `<b>`, `<i>`, `<code>`, `<s>`, `<u>`, `<pre>`
- Нет попыток использовать `<ul>`, `<ol>`, `<li>`, `<h1-h6>`
- Emoji корректно обрабатываются как многобайтовые символы
- Максимальная длина сообщения 4096 символов сохранена
