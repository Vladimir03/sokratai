## Цель

Сделать так, чтобы sokratai.ru отлично индексировался и ранжировался в Яндексе (РФ).

## Что уже есть

- Яндекс.Метрика 105827612 подключена (`index.html` + `useYandexMetrika`)
- `robots.txt` открыт всем ботам
- Канонический домен `https://sokratai.ru`, og-теги, JSON-LD отсутствует
- Sitemap-генератора нет, `sitemap.xml` не существует

## План (3 этапа)

### Этап 1. Технический SEO под Яндекс (в коде)

1. **Sitemap-генератор** `scripts/generate-sitemap.ts` + `predev`/`prebuild` хуки в `package.json`. Включить только публичные роуты: `/`, `/login`, `/register-tutor` (если есть посадочные), `/offer`, `/privacy-policy`, `/requisites`. Исключить `/tutor/*`, `/student/*`, `/chat`, `/admin`, `/p/*`, `/diagnostic` и т.п. — это закрытые/динамические.
2. `**public/robots.txt**` — добавить:
  - `Sitemap: https://sokratai.ru/sitemap.xml`
  - отдельный блок `User-agent: Yandex` + `Allow: /` + `Clean-param:` для типичных utm-меток (`utm_source&utm_medium&utm_campaign&utm_term&utm_content&yclid&gclid /`)
  - закрыть приватные разделы (`Disallow: /tutor/`, `/admin`, `/chat`, `/student/`, `/p/`)
3. **Яндекс-верификация** — добавить второй мета-тег `<meta name="yandex-verification" content="...">` в `index.html` (значение получим в Этапе 2 через Webmaster).
4. **JSON-LD `Organization` + `WebSite` с `SearchAction**` в `index.html` — помогает Яндексу формировать sitelinks и расширенный сниппет.
5. **Open Graph для VK** — добавить `og:locale="ru_RU"` (улучшает превью в Дзене/VK, которые часть Яндекс-экосистемы).

### Этап 2. Регистрация в Яндекс.Вебмастере (нужны твои действия)

После деплоя я не могу сам подтвердить домен — нужен твой клик. Я подготовлю пошаговую инструкцию:

1. Зайти на [https://webmaster.yandex.ru](https://webmaster.yandex.ru) → «Добавить сайт» → `https://sokratai.ru`
2. Выбрать способ подтверждения «Мета-тег» → скопировать `content="..."` → прислать мне → я вставлю в `index.html` и закоммичу.
3. После деплоя VPS — нажать «Проверить» в Вебмастере.
4. Загрузить sitemap: «Индексирование» → «Файлы Sitemap» → `https://sokratai.ru/sitemap.xml`.
5. «Региональность» → указать регион **Россия** (важно для геозависимых запросов «репетитор физики»).
6. «Информация о сайте» → указать официальное название «Сократ AI», описание, логотип.

### Этап 3. IndexNow (мгновенная индексация в Яндексе)

Яндекс поддерживает протокол IndexNow — пингуешь URL, он индексируется за минуты, не недели.

1. Сгенерировать ключ (32-символьный hex), положить файл `public/<key>.txt` с этим же ключом внутри.
2. Добавить простой скрипт `scripts/ping-indexnow.ts`, который после деплоя пингует `https://yandex.com/indexnow?url=https://sokratai.ru/&key=<key>` для главной + ключевых лендингов.
3. Опционально — `postbuild` хук, чтобы пинговать автоматически после каждого деплоя.

## Что вне scope (если захочешь — отдельной итерацией)
