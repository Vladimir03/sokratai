## План: Этап 2 — Яндекс.Вебмастер + IndexNow

### Шаг 1. Вставить Яндекс-токен в `index.html`
Заменить TODO-плейсхолдер на:
```html
<meta name="yandex-verification" content="33d83e09903d1840" />
```
(удалить комментарий `<!-- TODO Vladimir -->` и закомментированную строку)

### Шаг 2. Deploy на VPS
Изменение касается `index.html` → нужен `deploy-sokratai`:
```
ssh -i "$HOME/.ssh/sokratai_proxy" root@185.161.65.182
deploy-sokratai
```

### Шаг 3. Подтвердить домен в Яндекс.Вебмастере (ручное действие пользователя)
После деплоя:
1. Открыть https://webmaster.yandex.ru/sites/add/ → `sokratai.ru`
2. Выбрать META-tag способ → «Проверить»
3. Submit sitemap: `https://sokratai.ru/sitemap.xml`
4. В «Региональность» → Россия
5. В «Информация о сайте» → название «Сократ AI», описание

### Шаг 4. Пингануть IndexNow для первичной индексации
После того, как Webmaster подтвердил домен:
```
node scripts/ping-indexnow.mjs
```
Это уведомит Яндекс о 9 публичных URL — индексация начнётся за часы вместо недель.

### Шаг 5. Что дальше (после индексации 1-2 недели)
- Мониторить позиции в Webmaster по ключам: «AI проверка ДЗ», «платформа для репетиторов», «репетитор по физике»
